At this page you can keep all kind of notes for yourself. Furthermore, they can be categorized into certain sections.

The Notes function can be enabled or disabled per user in the user account settings --> "Wavelog Preferences" --> "Menu Options" --> "Show notes in the main menu."
<img width="624" height="217" alt="enable or disable the Notes function in the user settings" src="https://github.com/user-attachments/assets/0224ebf9-1c1c-47e7-8b10-84a507057835" />


When the Notes function is enabled, you'll find an additionally Callsign related Notes section in the Live QSO window, below the QSO logging details screen. This section is always minimized and needs to be expanded by clicking on the "double arrow" on the right side.
The Callsign Notes can be used to store special information about an specific callsign/OP. They won't be exported to an ADIF/external services.
