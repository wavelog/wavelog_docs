# User Accounts

As an Admin you can add additional Users to your installation.

The users table shows all already existing users with some additional information about each one.

## Creating an new User

To create a new user click `Create user` and fill out all necessary information in the form. You need to fill out at minimum all information in the first tab.

## Send Password Reset

You can send a Password Reset E-Mail to the User. Just click the Key Button in the "Password Reset Column".
